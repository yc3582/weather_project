# weather_project

## Overview

This Weather App allows users to compare the weather of two cities side by side. It fetches real-time weather data from an external API and displays it in a user-friendly format.

## Features

- Search and compare weather data for two cities.
- Display current temperature, weather conditions, and other relevant information.
- Simple and responsive user interface.

## Technologies Used

- **Backend:** Django
- **Frontend:** HTML, CSS
- **APIs:** OpenWeatherMap API (or any other weather data provider)
- **Environment Management:** `django-environ`
